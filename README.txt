This module covers basic and advanced of Spring Boot learnt over period of time.
The knowledge gained by completing this module will help in any Spring Boot application development.
Some features are-

- Hello World Simple Example
- CRUD operation using H2 DB
- Centralized exception handling using ControllerAdvice
- Internationalization
- Content Negotiation
- Actuator
- Versioning
